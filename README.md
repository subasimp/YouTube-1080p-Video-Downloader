Simple Youtube 1080p Video Downloader 

Lightweight video downloader for YouTube, built with Python, Streamlit, Pytubefix and FFmpeg
using separate video and audio files then merging them.

Features
- Download 1080p videos quick
- Automatic filename sanitization

Prerequisites
Before running this app, ensure that you have FFmpeg installed and added to your PATH directory
- [Download FFmpeg](https://ffmpeg.org/download.html)
- [How to add FFmpeg to PATH](https://www.youtube.com/results?search_query=how+to+add+ffmpeg+to+path)

Installation & usage
1. Clone this repository:
    ```bash
    git clone [https://github.com/subasimp/YouTube-1080p-Video-Downloader.git](https://github.com/subasimp/YouTube-1080p-Video-Downloader.git)

2. Install dependencies
    pip install -r requirements.txt

3. Run app
    streamlit run app.py
